
Get seulement pour /servers
Récupérer les infos des membres et des channels (servers.js)
    http://localhost:5000/api/servers/'serverID'

    ne nécessite rien

Envoyer une message dans un channel (messaging.js)

    http://localhost:5000/api/messaging/message/'channelID'

    nécessite dans l'URL le channelID

    nécessite dans le body :
    {
        "message" : ""
    }

Kick un membre (kick.js)
    http://localhost:5000/api/kick/user

    nécessite dans le body : 
    {
        "userID" : "",
        "guildID" : "",
        "reason" : ""
    }

Ban un membre (ban.js)
    http://localhost:5000/api/ban/user
